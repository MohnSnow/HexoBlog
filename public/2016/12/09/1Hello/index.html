一、HEXO命令

  1. hexo new “postName” -> 新建文章
  2. hexo clean -> 清缓存数据
  3. hexo generate -> hexo g 生成静态页面
  4. hexo server -> hexo s 启动本地服务，进行文章预览调试
  5. hexo deploy -> hexo d 部署到github
  6. hexo help -> 查看帮助
  7. hexo version -> 查看版本
